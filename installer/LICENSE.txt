QEMU Venus - License Information
=================================

This distribution bundles the following software:

QEMU (https://www.qemu.org/)
  License: GNU General Public License version 2 (GPL-2.0)
  With patches for: WHPX -cpu host passthrough, Venus virtio-gpu,
  120Hz EDID default refresh rate.

virglrenderer (https://gitlab.freedesktop.org/virgl/virglrenderer)
  License: MIT License
  With patches for: Windows Venus port, Win32 external memory shim,
  Intel driver push descriptor workaround, deferred object destruction.

Runtime libraries from MSYS2/UCRT64 (https://www.msys2.org/)
  Various open-source licenses (see individual library documentation).

Full source code and patches are available at the project repository.
